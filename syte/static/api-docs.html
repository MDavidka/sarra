Syte API

Version __VERSION__ — Base URL: /api

Authentication

All endpoints below require an API key. Create one in the Syte GUI under API Keys (navbar).

Header:  X-API-Key: syte_xxxxxxxxxxxxxxxx
   — or —
Header:  Authorization: Bearer syte_xxxxxxxxxxxxxxxx

Errors

HTTPerror codemeaning
401missing_api_keyNo X-API-Key or Bearer token sent
401invalid_api_keyToken revoked or wrong
400invalid_pathFile path escapes workspace sandbox
400create_failedProject could not be created (duplicate UUID, etc.)
404not_foundProject/workspace UUID does not exist
404delete_failedFile not found or cannot be deleted

Endpoints

GET/api/workspace_list

List all deployment workspaces.

Response

{
  "ok": true,
  "workspaces": [
    {
      "uuid": "my-app-a1b2c3",
      "name": "my-app",
      "status": "running",
      "deploy_type": "docker",
      "port": 3000,
      "git_url": "https://github.com/user/repo.git",
      "branch": "main",
      "workspace_path": "/var/lib/syte/workspaces/my-app-a1b2c3"
    }
  ]
}
POST/api/create_project

Create a project and start deployment in the background. Stream logs via /api/projects/{uuid}/logs/stream.

JSON body

fieldtyperequireddescription
namestringyesDisplay name
uuidstringnoCustom project ID (auto-generated if omitted)
git_urlstringnoFull git URL
git_providerstringnoShorthand host path, e.g. github.com/user/repo.git
branchstringnoDefault main
start_commandstringnoShell command if no Dockerfile
domainstringnoCustom HTTPS domain
env_varsobjectnoKey-value environment variables

Example

curl -X POST /api/create_project \
  -H "X-API-Key: syte_..." \
  -H "Content-Type: application/json" \
  -d '{
    "name": "nexhost",
    "git_url": "https://github.com/user/nexhost.git",
    "branch": "main"
  }'

Response

{
  "ok": true,
  "uuid": "nexhost-a1b2c3",
  "name": "nexhost",
  "port": 3000,
  "message": "Deploy started...",
  "stream_url": "/api/projects/nexhost-a1b2c3/logs/stream"
}
POST/api/issue_deploy

Re-run deploy for an existing project (git pull + rebuild/restart).

JSON body

{ "uuid": "nexhost-a1b2c3" }
POST/api/execute_command

Run a shell command inside a workspace (sandboxed to workspace directory).

JSON body

fieldtyperequireddescription
uuidstringyesProject UUID
commandstringyesShell command
cwdstringnoRelative dir, default app
timeoutintnoSeconds, default 120, max 600

Response

{ "ok": true, "exit_code": 0, "output": "..." }
POST/api/delete_file

Delete a file inside a workspace.

JSON body

{ "uuid": "nexhost-a1b2c3", "path": "app/temp.txt" }
POST/api/upload_file

Upload a file (multipart form).

Form fields

fielddescription
uuidProject UUID
pathRelative destination path, e.g. app/config.json
fileFile binary
curl -X POST /api/upload_file \
  -H "X-API-Key: syte_..." \
  -F "uuid=nexhost-a1b2c3" \
  -F "path=app/.env" \
  -F "file=@.env"

Real-time deployment logs (SSE)

GET/api/projects/{uuid}/logs/stream

Server-Sent Events stream of build and runtime logs. No auth required from browser GUI; include X-API-Key for programmatic access.

Event format

data: {"type": "build", "text": "Step 1/21 : FROM node:22-alpine"}
data: {"type": "container", "text": "Server listening on 3000"}
data: {"type": "ping"}

JavaScript example

const es = new EventSource('/api/projects/UUID/logs/stream');
es.onmessage = (e) => {
  const msg = JSON.parse(e.data);
  if (msg.text) console.log(msg.text);
};
