Sycord API

Sycord Deployer API

v__VERSION__ · Base: /sycord/api · spec.json · Auth: X-API-Key

Connect Sycord websites and external projects to Syte. project_connect auto-creates a workspace and subdomain {slug}.sycord.site.

Stacks

  • nextjs (default) — Next.js 14 + Dockerfile standalone
  • python — FastAPI + uvicorn
  • javascript — Express Node.js

Workflow

  1. POST /sycord/api/project_connect
  2. POST /sycord/api/upload — add files
  3. POST /sycord/api/issue_deployment
  4. GET /sycord/api/container_get?uuid=
POST/sycord/api/project_connect

Creates workspace, scaffolds stack, assigns https://{slug}.sycord.site.

{
  "name": "testproject",
  "stack": "nextjs",
  "env_vars": {}
}
{
  "ok": true,
  "uuid": "testproject-a1b2c3",
  "domain": "testproject.sycord.site",
  "url": "https://testproject.sycord.site",
  "stack": "nextjs"
}
GET/sycord/api/container_get?uuid=

Returns Docker container state, image, and URLs.

{ "ok": true, "uuid": "...", "container_name": "syte-...", "running": true, "url": "https://..." }
POST/sycord/api/upload

Multipart: uuid, path (e.g. app/app/page.tsx), file

POST/sycord/api/domain
{ "uuid": "testproject-a1b2c3", "domain": "custom.sycord.site" }
POST/sycord/api/issue_deployment

Docker build + container start. Stream logs via /api/projects/{uuid}/logs/stream?live=1

{ "uuid": "testproject-a1b2c3" }
